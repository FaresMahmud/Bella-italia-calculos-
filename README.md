---

## ▶️ Como rodar o Python localmente

Requer Python 3 instalado. No terminal:

```bash
python Bella-italia-calculos.py
```

O script imprime no terminal a classificação dos gastos, CVU, custo fixo total, custo por pote, margem de contribuição, ponto de equilíbrio e lucro mensal.

---

## 📊 O que está no relatório

- Classificação correta de cada gasto (Custo Variável, Custo Fixo, Despesa Fixa, Despesa Variável)
- Cálculo do Custo Variável Unitário (CVU)
- Cálculo do Custo Fixo Total e Custo Total por Pote
- Margem de Contribuição Unitária
- Ponto de Equilíbrio Contábil (PEC)
- Simulador interativo — ajuste volume e preço e veja os indicadores em tempo real
- Diagnóstico estratégico para o Sr. Lorenzo
